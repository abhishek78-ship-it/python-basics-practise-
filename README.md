# Python Basics Practice 🐍

Welcome to my Python practice repository!

This repository contains beginner-friendly Python programs that I created while learning Python from scratch. The goal is to build a strong foundation in programming by practicing basic concepts and solving simple problems.

## 📚 Topics Covered

- Variables and Data Types
- Input and Output
- Conditional Statements (`if`, `elif`, `else`)
- Arithmetic Operators
- Comparison Operators
- Loops (`for`, `while`)
- Functions
- Lists and Tuples
- Dictionaries and Sets
- String Manipulation
- Basic Problem Solving
